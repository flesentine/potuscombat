# POTUS COMBAT

A tiny browser fighting-game prototype: 16-bit arcade style, presidential fighters, generated pixel-art stage/sprites, and simple keyboard combat.

## Play

Open `index.html` in a browser.

Controls:

- `WASD` move
- `J` punch
- `K` kick
- `I` Lincoln hat throw
- `L` special
- `Space` block
- `Enter` start

## Assets

The White House stage and Washington animation cells were generated as bitmap assets, then integrated into the canvas renderer.
